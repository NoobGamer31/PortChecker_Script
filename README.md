# PortChecker_Script
# VERSÃO ATUAL: 1

## ❓| O que faz?
Tal como o nome indica o código serve para verificar uma porta do computador. Este código de C# utiliza o comando da CMD chamado **netstat**. Esse curto comando permite verificar qualquer porta do sistema, retornando algumas informações acerca dela. Com este pequeno script, é possível fazer o mesmo, mas num clicar de botão e introdução da porta.

## 📖 | Como usar?
Para usar, basta descarregar e abrir o .exe que se encontra no release do Github (pode também criar um atalho quando baixado). Após abrir, irá surgir a tela do CMD, nela você digita o número da porta e então espera o código rodar! 

## ⚠️ | Nota
- Em caso de erro de digitação, a versão 1 não possui reload automático do programa ou permitir digitar novamente, nem suporte rodar várias vezes no mesmo clique (Por exemplo: Cada execução só verá uma porta). Confira ou aguarde versões futuras para essa atualização.
- Necessita do .NET 10 instalado na máquina para a versão 1 do script
