# PortChecker_Script
# VERSÃO ATUAL: 2

## ❓| O que faz?
Tal como o nome indica o código serve para verificar uma porta do computador. Este código de C# utiliza o comando da CMD chamado **netstat**. Esse curto comando permite verificar qualquer porta do sistema, retornando algumas informações acerca dela. Com este pequeno script, é possível fazer o mesmo, mas num clicar de botão, seleção do que deseja fazer e introdução da porta.

## 📖 | Como usar?
Para usar, basta descarregar e abrir o .exe que se encontra no release do Github (pode também criar um atalho quando baixado). Após abrir, irá surgir a tela do CMD, nela você é apresentado ao menu do script, então basta escolher a opção que deseja executar. De momento apenas é possível verificar uma porta, ou sair do script.

## 🎉 | Ideias futuras
Como descrito no menu de início da versão 2, o script poderá ser atualizado para permitir a verificação de inúmeras portas, ou de um intervalo de portas. Futuramente está pensado em ser adicionado um suporte a multi-línguas, nomeadamente Português e Inglês, permitindo um uso do script a nível quase mundial.

## ⚠️ | Notas
- Em caso de erro de digitação, a versão 1 não possui reload automático do programa ou permitir digitar novamente, nem suporte rodar várias vezes no mesmo clique (Por exemplo: Cada execução só verá uma porta). Versão 2 e acima possuem essa função.
- Necessita do .NET 10 instalado na máquina para a versão 1 do script. Versão 2 e acima não.
